Responsive Web Design

Responsive Web Design is technology whose aim is to make web design look perfect across all types of devices, from smartphones to 27" screens, achieving optimum display by resizing and restructuring elements according to the grid in order to take maximum advantage of space.

With devices like smartphones and tablets used so widely today, we must ensure that our websites look perfect on all of them, hence the need to include this technology, and when designing we must account for all formats.
