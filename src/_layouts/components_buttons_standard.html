Neutral

Neutral styling is the default and should be applied whenever there is no strong reason to do otherwise


Positive

Positive buttons are reserved for particularly important actions. There should only be one positive button per context


Negative

Negative buttons imply a significant negative action. They should only be used where user actions have serious, lasting effects that need to be clearly explained.


Subdued

Subdued buttons are not the default. They are used to intentionally downplay secondary actions like “Back” or “Cancel”.
